# Machine Learning Internship Projects

## Overview

This repository contains three Machine Learning and Natural Language Processing (NLP) projects completed as part of the Machine Learning Internship. Each project demonstrates different concepts of data preprocessing, model building, prediction, and evaluation using Python and Scikit-learn.

---

# Project 1: Sales Prediction Using Machine Learning

## Objective

Develop a machine learning model to predict future sales using historical advertising data.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Correlation analysis
* Linear Regression model
* Sales prediction
* Model evaluation using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)
  * R² Score
* Data visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

# Installation

1. Install Python (3.10 or later recommended).
2. Clone or download this repository.
3. Install the required libraries:

```bash
python -m pip install -r requirements.txt
```

If you do not have a `requirements.txt` file, install the libraries manually:

```bash
python -m pip install pandas numpy matplotlib seaborn scikit-learn
```
